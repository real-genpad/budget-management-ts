Это переписанный на TS "Budget-management-JS"(пока в процессе)
Сервер учебный, данные удаляются каждые 24 часа

В папке static хранятся статичные элементы (картинки и шрифты)

Layout (sidebar) хранится в отдельном файле в темплейтах и в него подгружаем остальной контент.
Шаблон состоит из левого сайдбара и блока, в который вставляется основной контент страницы (<div class="content-wrapper">),
а сам layout вставляется в <div id="content"></div>

Все стили, которые являются одинаковыми для разных страниц (это контейнер, кнопки и заголовки, категории), собраны в общем
файле styles. В папке income-expenses находятся стили для страниц Доходы и расходы и с ними связанные. В папке templates также созданы отдельные папки
для шаблонов и скриптов этих групп страниц. В папке components стили для остальных страниц

В папке templates хранятся шаблоны html всех страниц, которые подставляются в файл index.html

В папке components хранятся js файлы с классами, соответствующие каждому роуту, в которых прописана основная логика
каждой страницы

В папке utils находятся утилиты:
    auth-utils - работа с токенами и авторизация
    http-utils - работа с запросами

В папке config хранятся данные в зависимости от энвайронмента (например, на продакшене будет другой url)
     config - содержит url для подстановки в запросы

В папке services ноходится файл date-filter - содержит логику работы фильтров выбора дат

Редирект не через window.location.href, а через функцию в файле router.ts (openNewRoute)
